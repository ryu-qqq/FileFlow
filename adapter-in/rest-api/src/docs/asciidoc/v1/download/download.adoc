[[download-api]]
= 다운로드

외부 URL의 파일을 다운로드하여 S3에 저장하는 비동기 다운로드 태스크 API입니다. 태스크 생성 시 SQS 큐에 작업이 등록되며 워커가 비동기로 처리합니다.

`/api/v1/downloads`

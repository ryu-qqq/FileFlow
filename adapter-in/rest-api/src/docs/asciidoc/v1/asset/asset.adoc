[[asset-api]]
= 에셋

업로드가 완료된 파일의 메타데이터를 관리하는 API입니다. Asset은 세션 완료 또는 다운로드 완료 시 자동으로 생성되며, 외부에서 직접 생성할 수 없습니다.

`/api/v1/assets`

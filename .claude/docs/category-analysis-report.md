## 1. 카테고리 계층 구조 (Depth) 비교

| 플랫폼 | Depth | 구조 | 총 카테고리 수 |
|--------|-------|------|---------------|
| **네이버 쇼핑** | 4단계 | 대분류 > 중분류 > 소분류 > 세분류 | ~5,000개 |
| **쿠팡** | 4단계 | 대분류 > 중분류 > 소분류 > 세분류 | ~3,000개+ |
| **아마존** | 5~7단계 | 계층적 Browse Node 시스템 | 25,000+ 하위 카테고리 |
| **G마켓/옥션** | 4단계 | ESM 카테고리 (N:N 매핑) | ~4,000개 |

## 2. 주요 트렌드 분석

### 2.1 그룹상품 (Variant Group) 도입
- **네이버 2024~2025**: 기존 190개 → 1,000개 카테고리로 확장
- 색상, 크기 등 판매 옵션을 "그룹 단위"로 관리
- 리뷰/인기도 집계의 통합
- **시사점**: 개별 SKU가 아닌 "상품군" 단위 관리 필요

### 2.2 카테고리 세분화 트렌드
- 뷰티/건강: 기능성 세분화 (리프팅밴드, 필터샤워기, 건강기능식품)
- 생활용품: 트렌드 반영 (센서 휴지통, 다꾸스티커)
- 스포츠: 종목별 세분화 (축구/농구/야구 유니폼)

### 2.3 아마존의 Product Type + Browse Node 이원화
- **Product Type**: 내부 분류용 (속성/템플릿 결정)
- **Browse Node**: 고객 노출용 (검색/필터)
- **시사점**: 관리용 분류와 노출용 분류 분리 가능

## 3. 상품정보제공고시 (법적 요구사항)

공정거래위원회 고시에 따른 약 **35개 품목** 분류:

| CategoryGroup | 고시 품목명 | 필수 정보 항목 |
|--------------|------------|---------------|
| CLOTHING | 의류 | 제조자, 소재, 치수, 세탁방법, 제조국 |
| SHOES | 구두/신발 | 제조자, 소재, 굽높이, 제조국 |
| BAGS | 가방 | 종류, 소재, 크기, 제조자, 제조국 |
| ACCESSORIES | 패션잡화 | 종류, 소재, 치수, 제조자 |
| COSMETICS | 화장품 | 용량, 주요성분, 사용기한, 사용방법 |
| JEWELRY | 귀금속/보석 | 소재, 순도, 중량, 제조자 |
| WATCHES | 시계 | 종류, KC인증, 방수등급, 제조자 |
| DIGITAL | 영상가전/가정용전기제품 | KC인증, 정격전압, 소비전력, 제조자 |
| FURNITURE | 가구 | 품명, 크기, 재질, 색상, 제조자 |
| SPORTS | 스포츠용품 | 품명, 재질, 사이즈, 제조자 |
| BABY_KIDS | 영유아용품 | KC인증 필수, 사용연령, 주의사항 |
| ETC | 기타 재화 | 품명, 모델명, 제조자 (최소 정보) |

## 4. 현재 market DB 카테고리 현황

### 4.1 통계 (마이그레이션 전)
- 총 카테고리 수: ~200개
- depth 분포: depth 1 (8개), depth 2 (~40개), depth 3 (~150개)
- CategoryGroup 분류:
  - CLOTHING: 패션 관련
  - ETC: 194개 (대부분 미분류 상태)

### 4.2 V12 마이그레이션으로 수정될 내용
```sql
-- DIGITAL: 디지털/가전 (id=4 하위)
-- FURNITURE: 가구 (id=7 하위)
-- SPORTS: 스포츠 (id=6 하위)
-- BABY_KIDS: 유아동 (id=8 하위 중 84)
```

## 5. 카테고리 세분화 제안

### 5.1 Depth 4 추가 대상

| 현재 경로 (depth 3) | 제안 세분화 (depth 4) | CategoryGroup |
|--------------------|----------------------|---------------|
| 패션 > 여성의류 > 티셔츠 | 반팔/긴팔/민소매/크롭 | CLOTHING |
| 패션 > 여성의류 > 니트/스웨터 | 라운드넥/브이넥/터틀넥/가디건 | CLOTHING |
| 패션 > 신발 > 스니커즈 | 러닝화/캔버스/하이탑/로우탑 | SHOES |
| 패션 > 가방 > 백팩 | 데일리/등산/노트북/미니 | BAGS |
| 디지털 > TV > TV | LED/OLED/QLED/미니LED | DIGITAL |
| 가구 > 침대 > 침대 | 싱글/더블/퀸/킹/패밀리 | FURNITURE |
| 스포츠 > 피트니스 > 운동복 | 상의/하의/세트/레깅스 | SPORTS |

### 5.2 CategoryGroup 재분류 필요 카테고리
- 화장품/뷰티 → COSMETICS
- 주얼리/시계 → JEWELRY, WATCHES
- 유아동 전체 → BABY_KIDS
- 생활용품 → ETC 또는 새 그룹
